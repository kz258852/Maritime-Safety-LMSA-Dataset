# Maritime Safety LMSA Dataset 

This repository provides sample datasets used in the LMSA framework for maritime safety management. 

## 📌 Overview
The dataset supports research on multimodal perception and LLM-based reasoning for maritime safety, including:
- AIS trajectory data
- Visual perception annotations
- Radar perception data
- Instruction-response pairs for LLM fine-tuning

To facilitate reproducibility and experimental validation, partial datasets used in this study are publicly released through Baidu Netdisk links, covering AIS trajectories, visual perception samples, and radar sensing data.

## 📊 Dataset Components

### 1. AIS Data
- Attributes: MMSI, timestamp, latitude, longitude, speed, heading
- Preprocessing: noise filtering, interpolation, anomaly removal
- Supports trajectory analysis, anomaly detection, and vessel behavior understanding

## 📍 Notes
- The released dataset is anonymized and partially sampled due to privacy constraints.
- Some original maritime monitoring data have been desensitized and compressed for public release.
- Full-scale datasets and additional annotations are available upon reasonable academic request.
- The provided Baidu Netdisk resources are intended solely for non-commercial academic research purposes.
